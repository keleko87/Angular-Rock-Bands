
 Explicación de como he afrontado algunos puntos que requería el ejercicio y problemas encontrados.


1 - El ejercicio pedia que la HOME tuviera el lista de las demás páginas

    -  La HOME en el proyecto en index.html, el cual contiene los elementos comunes a todas las páginas (head, footer...)


2 - El ejercicio pedía crear 3 páginas para cada una de las bandas (Rolling, Beatles y Queen)

    -  En este punto he entendido que las 3 páginas de bandas, debía ser escalable, es decir que haya un template que liste todas las bandas (home.list.html)
    y una template detalle (home.detail.html). Decicí utilizar el "Abstract State" para Home y listar las bandas
    Aqui encontré el problema de que la URL de home.detail.html es por ID y el enunciado pedía por URL. No fui capaz de hacerlo por URL con $stateParams.
    Imagino que si que es posible hacerlo.

    - Las bandas estan contenidas en un array. Lo suyo habría sido que hubiera creado un JSON y leerlo.


3 - SEO

    - De la manera que he creado el listado de bandas, en vez de crear una template Html
      he limitado la opción de poder dar a cada banda sus valores para el SEO.
